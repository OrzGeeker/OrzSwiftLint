# OrzSwiftLint

The Unite Lint Configuration for all swift projects of OrzGeeker

## Usage

```swift
    .target(
        name: "OrzSwiftLint",
        plugins: [
            "OrzSwiftLintBuildToolPlugin"
        ]
    )
```
